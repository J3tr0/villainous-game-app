## Installazione

```bash
# Clona il repository
git clone https://github.com/J3tr0/villainous-companion.git

# Entra nella directory
cd villainous-companion

# Installa le dipendenze
pnpm install

# Avvia il server di sviluppo
pnpm dev
```
